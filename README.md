# Running app

## Frontend

To run Frontend, you will need Angular 6 installed.

```shell
npm install -g @angular/cli
```

Then, you will be able to run:

```shell
(cd FrontEnd && ng serve --open)
```

## Backend

To run Backend, you will need:
    Ruby 2.3.1,
    Rails 5,
    Sqlite3,
installed.

```shell
(cd BackEnd && bundle && rails server)
```
